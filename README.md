# PDF Chatbot

Ein KI-gestützter Chatbot, der Fragen zu PDF-Dokumenten beantwortet.

## Entstehung
Entwickelt für meinen Bruder, der als Schüler täglich mit 
mehreren PDF-Skripten arbeitet. Statt alles manuell zu 
durchsuchen, kann er jetzt direkt Fragen stellen.

## Features
- PDF per Drag & Drop hochladen
- Fragen im Chat stellen
- Kontext wird über mehrere Nachrichten gemerkt
- Läuft komplett im Browser

## Technologien
- Vanilla HTML/CSS/JavaScript
- Anthropic Claude API (claude-sonnet-4-20250514)
